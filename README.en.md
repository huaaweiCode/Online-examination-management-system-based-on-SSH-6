# 基于SSH的在线考试管理系统6

#### Description
基于SSH的在线考试管理系统8分为三种角色：

**管理员**：用户管理（管理员、教师和学生信息管理）、试题管理（单选题、多选题、判断题、问答题）、学科管理、学院管理

**教师**：试卷管理（生成试卷、查看试题）、考试阅卷管理

**学生**：考试管理（查看所有试卷和考试）、成绩管理
